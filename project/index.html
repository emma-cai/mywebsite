FreeParser

FreeParser is a system that trains on one domain and one set of predicate and constant symbols, and then can parse sentences for any new domain, including sentences that refer to symbols never seen during training.

In our first attempt, FreeParser uses a domain-independent architecture to automatically identify sentences relevant to each new database symbol, which it uses to supplement its manually-annotated training data from the training domain. In cross-domain experiments, FreeParser can parse sentences for which it has seen comparable unannotated sentences with an F1 of 0.71.

In the second attempt, FreeParser scales semantic parsing to large-scale databases based on a reduction to standard supervised training algorithms, schema matching and pattern learning. Leveraging techniques from each of these areas, FreeParser is capable of parsing questions with an F1 that improves by 0.42 over a purely-supervised learning algorithm.

Freebase

Freebase is a large collaborative knowledge base consistng of metadata composed mainly by its community members. It is an online collection of structured data havested from many sources, including individual wiki contributions. Freebase aims to create a global resource which allows people (and machines) to access common information more effectively. It was developed by the American software company Metaweb and has been running publicly since March 2007. Metaweb was acquired by Google in a private sale announced July 16, 2010

Freebase data is available for free for commercial and non-commercial use under a Creative Commons Attribution License, and an open API, RDF endpoint, and database dump are provided for programmers.
